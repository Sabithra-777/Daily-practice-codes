# Day 90 - DSA Practice

## Problem:
Check whether a matrix is a Magic Square

## Approach:
- Calculate the sum of the first row
- Compare every row sum
- Compare every column sum
- Compare both diagonal sums
- If all sums are equal, it is a Magic Square

## Example:
Input:
8 1 6
3 5 7
4 9 2

Output:
Magic Square

## Language:
C
