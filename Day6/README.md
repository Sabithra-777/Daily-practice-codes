# Day 6 - DSA Practice

## Problem:
Search an element using Binary Search

## Approach:
- Works only on sorted array
- Find middle element
- Compare with key
- If equal → found
- If smaller → search right
- If larger → search left

## Example:
Input: 10 20 30 40 50  
Search: 40  
Output: Found at position 4
