# Day 91 - DSA Practice

## Problem:
Create and display a singly linked list

## Approach:
- Create nodes dynamically using malloc()
- Insert each node at the end
- Traverse the linked list and print each node

## Example:
Input:
5
10 20 30 40 50

Output:
10 -> 20 -> 30 -> 40 -> 50 -> NULL

## Language:
C
