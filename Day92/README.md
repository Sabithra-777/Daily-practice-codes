# Day 92 - DSA Practice

## Problem:
Insert a node at the beginning of a singly linked list

## Approach:
- Create a new node
- Point its next to the current head
- Update head to the new node
- Display the updated linked list

## Example:
Input:
20 30 40

New Node:
10

Output:
10 -> 20 -> 30 -> 40 -> NULL

## Language:
C
