# Day 87 - DSA Practice

## Problem:
Find the smallest and largest element in a matrix

## Approach:
- Initialize smallest and largest with the first element
- Traverse the matrix
- Update smallest and largest whenever needed

## Example:
Input:
3 8 1
9 4 6
7 2 5

Output:
Smallest Element = 1
Largest Element = 9

## Language:
C
